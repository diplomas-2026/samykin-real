# Техническая спецификация

## Тема

Разработка программного обеспечения для распределения и выдачи денежных выплат в ИП Самыкин.

## Роли

- Администратор
- Бухгалтер
- Сотрудник

## Страницы

- `/` — публичная главная страница продукта
- `/login` — вход
- `/dashboard` — панель бухгалтера со статистикой и графиками
- `/profile` — профиль пользователя
- `/users`, `/users/new`, `/users/:id` — управление пользователями
- `/employees` — список сотрудников
- `/payouts`, `/payouts/new`, `/payouts/:id` — выплаты
- `/my-payouts`, `/my-payouts/:id` — выплаты сотрудника

## Статусы выплаты

- `CREATED` — Создана
- `PREPARED` — Подготовлена
- `PAID` — Выдана
- `CANCELLED` — Отменена

## AI-функция

LLM используется для генерации комментариев и пояснений к денежным выплатам.

- технология: Spring AI + GigaChat
- токен GigaChat хранится в env
- администратор может настраивать только стиль генерации
- системный prompt скрыт
- дневной лимит системы: 20 000 токенов
- лимит общий на всю систему
- сброс лимита: каждый день в 00:00 по Самаре

## Excel-отчеты

- список выплат за период
- список выданных и невыданных выплат
- отчет по сотрудникам
- сводный отчет по суммам и статусам
